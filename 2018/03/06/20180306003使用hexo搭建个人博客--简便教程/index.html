使用hexo搭建个人博客--简便教程

2018-03-06

前置环境配置

本教程的前置环境是，在windows/mac中 安装

  • git环境
  • node环境

以及在github中配置好ssh相关部署（此部分省略）

快速搭建本地环境

在本地环境中新建一个文件夹，名为Hexo，存放位置以这个为例（E:\blog\Hexo）
在该文件夹下右击 git bash

此部分的cnpm是安装了淘宝镜像，若没有安装请使用npm命令替代。
当然你可以直接安装淘宝镜像就可以使用，运行速度更快

安装淘宝镜像

快速部署项目

这时会生成一个本地链接
http://localhost:4000/.

更改主题样式

在hexo官网中选择喜欢的主题 https://hexo.io/themes/

回到你本地的路径（E:\blog\Hexo\themes）
在该文件夹下右击 git bash

以这个主题为例（你可以随便更换主题，clone的链接可以是git也可以是https）

clone之后回到，打开_config.yml
更改

更改
